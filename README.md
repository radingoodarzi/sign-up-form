## 📁 Project Setup

1. Create two folders named **html** and **css**.
2. Move the HTML files into the **html** folder:
   - `login.html`
   - `sing up.html`
3. Move the CSS files into the **css** folder:
   - `loginstyle.css`
   - `singup.css`

The final project structure should look like this:

```text
Project/
├── html/
│   ├── login.html
│   └── sing up.html
└── css/
    ├── loginstyle.css
    └── singup.css
```
